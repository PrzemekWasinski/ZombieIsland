Zombie Island

Zombie Island is a 2D survival game where the player finds himself stranded on an island and has to fight waves of zombies which get increasingly more difficult. You need to survive for as long as possible and defeat as many zombies as you can to achieve a score which is displayed on the rankings page.

The game features a health system for the player and the zombies, fighting, items dropped by the zombies that the player can use, 2D pixel-art style graphics and a round system where each round becomes more difficult for the player. The game's controls can be seen on the left, restart and quit will only become available once the game is over. Be aware your score will not save if you go to another page before the game ends!

Before you start playing make sure to register and login to your account so that your highscore appears on the rankings page.
