Welcome! This is a work in progress game. You're welcome to try it out although it may have problems/bugs at this time.

Zombie Island is an online RPG web game, players can make an account, fight enemies, chat with other players,
explore, collect items by defeating enemies and destroying objects around the map and more coming soon!

Controls:

  • W A S D: Walk
  • Space: Hit enemies/damage objects
  • I: Open/close inventory
  • T: Start typing a message
  • Escape: Exit inventory/Cancel typing a message
  • Enter: Send message
