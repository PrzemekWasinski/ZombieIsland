Welcome!

This is a work in progress game (Currently Desktop only). You're welcome to try it out, but please be aware of potential bugs. Zombie Island is an online RPG where you can fight enemies, chat with players, explore the world, and collect items.

Controls:

  • W A S D: Walk
  • Space: Attack
  • E: interact
  • I: Open/close inventory
  • T: Start typing a message
  • Enter: Send message
  • Escape: Close menu / Cancel typing
